Múltiples perspectivas

Las tres instancias del análisis

Formación

Comprende los procesos educativos, capacitación y desarrollo de competencias para personas con discapacidad.

Intermediación

Analiza los mecanismos, actores y estrategias que conectan la formación con oportunidades de empleo.

Empleo

Examina las condiciones, desafíos y realidades del empleo propiamente tal para personas con discapacidad.

Fuentes y datos

Consolidado a partir de análisis de entrevistas anonimizadas. Se utilizó codificación temática para identificar fragmentos relevantes y su distribución en las tres instancias (Formación, Intermediación, Empleo).

Para consultar la distribución detallada de entrevistas por instancia: Ver datos de fuentes →

Herramientas y versión

Versión v4: Se incluyen explícitamente condensaciones y temas previamente excluidos para evitar pérdida de casos. Esta iteración añade cobertura completa del corpus de análisis.

Visualizaciones generadas con: Plotly — Biblioteca de código abierto para gráficos interactivos. Los diagramas Sankey permiten explorar flujos de datos complejos de manera intuitiva sin sacrificar fidelidad analítica.

Uso sugerido

Exploración rápida para seminarios, discusiones de comité, preparación de informes ejecutivos y presentaciones en espacios de política pública. Las visualizaciones están optimizadas para lectura cualitativa.
