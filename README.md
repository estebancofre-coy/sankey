# Sankey - Temas Transversales

Visualizaciones de diagramas Sankey para análisis de temas transversales.

## Contenido

- **index.html**: Página principal con enlaces a las visualizaciones
- **sankey_General_inst_tema.html**: Sankey general: Instancia → Tema
- **sankey_Formación_cond_tema.html**: Formación: Condensación → Tema
- **sankey_Intermediación_cond_tema.html**: Intermediación: Condensación → Tema
- **sankey_Empleo_cond_tema.html**: Empleo: Condensación → Tema

## Despliegue en Vercel

Para desplegar este proyecto en Vercel:

1. Instala Vercel CLI (si no lo tienes):
   ```bash
   npm install -g vercel
   ```

2. Desde el directorio del proyecto, ejecuta:
   ```bash
   vercel
   ```

3. Sigue las instrucciones en la terminal para conectar tu cuenta y desplegar.

Alternativamente, puedes:
- Importar este repositorio directamente desde [vercel.com](https://vercel.com)
- Conectar tu cuenta de GitHub con Vercel para despliegue automático

## Ver localmente

Para ver las páginas localmente, abre `index.html` en tu navegador o usa un servidor HTTP simple:

```bash
python3 -m http.server 8080
```

Luego visita `http://localhost:8080`
